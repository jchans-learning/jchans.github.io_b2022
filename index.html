重讀一些以前喜歡的文章，發現自己讀東西的時候時常會只讀到自己想讀到的東西，而忽略了一些細節，也就是我讀進去的其實不是完全的、不是非常準確的作者的原意，這種狀況有點危險，會忽略掉很重要的事情。注意到了，試著改善吧。

突然想想，把自己閒置的舊筆電拿回來重灌就好了，單純用 ubuntu 的話，寫點 code 也沒啥效能的需求，應該夠用。只是老電腦，風扇有點吵、又熱。

如同 K 以前說的，重灌或買新電腦，最麻煩的其實是「個人化」這件事。這其實也是「雲端」這個概念前幾年會紅起來的原因。但這個部份，這幾年來並沒有什麼改善。

還是要去用 atom ？據說是非常吃資源，但是真的有些功能很懶人... 比起 vim... 。
